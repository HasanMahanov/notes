# ENUM4LINUX UCUN KOMMANDLAR
    "enum4linux [options] ip"

TAG            FUNCTION

-U             get userlist

-M             get machine list

-N             get namelist dump (different from -U and-M)

-S             get sharelist

-P             get password policy information

-G             get group and member list

-a             all of the above (full basic enumeration)
